## 项目简介

版本：0.1

项目地址：https://github.com/wangwangwang23333/Five-Son-s-Chess-Online

介绍：

一个可以支持联机进行五子棋的游戏

目前支持功能：等待其他玩家加入、查看上一步下棋位置

未来预期加入功能：悔棋、聊天、选择黑白方、积分功能



## 代码环境

编译器：vscode

语言：python

版本：3.7+

库：pygame、socket等



## 项目方法

1.将server.py中ip地址更改为服务器的私网ip

2.将playchess.py中ip地址更改为服务器的公网ip

3.打开服务器

4.两端即可同时开始游戏

注：可以通过Pyinstall库部署为.exe文件
